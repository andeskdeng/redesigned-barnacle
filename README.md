# Prime Number Explorer 🔢

A C++ project exploring prime number generation, mathematical patterns, and algorithm experimentation.

## Project Overview

This project was built to experiment with number theory concepts and algorithm design using C++.

The project explores:

- Prime number generation
- Mathematical logic
- Algorithm efficiency
- Computational thinking
- C++ programming

## Features

✅ Prime number calculations  
✅ Mathematical experimentation  
✅ C++ implementation  
✅ Algorithm exploration  

## Technologies Used

- C++
- Standard Library (STL)

## Installation

Clone repository:

```bash
git clone https://github.com/andeskdeng/redesigned-barnacle.git
cd redesigned-barnacle
```

Compile:

```bash
g++ main.cpp -o prime
```

Run:

```bash
./prime
```
